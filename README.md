# Biblioteca Multimedia

Aplicación web para gestionar una biblioteca multimedia personal, permitiendo registrar, buscar y administrar libros, películas y música en un catálogo digital.

# Tecnologías utilizadas:

## Frontend
- React
- HTML
- CSS
- JavaScript
- Axios

## Backend
- Node.js
- Express

## Base de datos
- PostgreSQL

## Deploy
- Vercel para el Frontend  
- Render para el Backend

# Funcionalidades

- Inicio de sesión
- Registro de usuario
- Menú principal para acceder al catalogo
- CRUD completo de items:
  - Crear
  - Leer
  - Actualizar
  - Eliminar
- Buscador de catálogo
- Filtro por categorías
  - Libros
  - Películas
  - Música
- Panel de estadísticas
- Interfaz moderna y responsive
- Mensajes cuando no hay resultados en búsquedas

# Características destacadas

- Arquitectura Frontend / Backend / Database
- Comunicación mediante API REST
- Interfaz moderna desarrollada con React
- Gestión completa de catálogo multimedia
